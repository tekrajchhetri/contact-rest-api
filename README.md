# Contact Rest API

REST API for contact management with token based authentication.


### Prerequisites

Python 3.X

Django

Django REST
```
pip install package
```


## Running
If 8000 port is used then provide alternative port number.
```
cd root-directory
python manage.py runserver [portnumber]

```
